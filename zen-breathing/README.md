# Zen Breathing (禅ブリージング) 🧘‍♂️

5分で心を落ち着かせる、シンプルな呼吸法アプリです。
美しいビジュアルとリズムに合わせて、リラックスしたひとときを過ごしましょう。

## 特徴 ✨

*   **直感的なデザイン**: ダークモードを基調とした、目に優しい美しいグラデーションとグラスモーフィズムデザイン。
*   **4-4-4 ブリージング**: 吸う(4秒)・止める(4秒)・吐く(4秒)・止める(4秒)のボックスブリージング法（または類似のリズム）を採用し、効果的なリラックスを促します。
*   **視覚的なガイド**: 呼吸に合わせて伸縮する円のアニメーションで、直感的に呼吸のタイミングを掴めます。
*   **シンプル**: ワンタップでセッションの開始・終了が可能です。

## 使い方 🚀

1.  このリポジトリをクローンします。
    ```bash
    git clone https://github.com/your-username/zen-breathing.git
    ```
2.  `index.html` をブラウザで開くだけで動作します。ローカルサーバーは必須ではありませんが、VS Codeの "Live Server" などを使うと便利です。
    ```bash
    # 例: openコマンドで開く (macOS)
    open index.html
    ```
3.  「Start Session」ボタンを押して、リラックスを開始してください。
