# Skill: Deploy Hono API

## Goal
Deploy the Hono API backend located in the `hono-api` directory to Cloudflare Workers/Pages.

## Instructions
1. Change directory to `hono-api`.
2. Run `npm run deploy` to complete the deployment to Cloudflare.

## Verification
- Confirm that the deployment was successful by checking the output for the Cloudflare Worker URL.
