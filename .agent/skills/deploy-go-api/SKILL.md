# Skill: Deploy Go API to VPS

## Goal
Deploy the Go API located in the `go-api` directory to a Hetzner VPS using Kamal.

## Instructions
1. Change directory to `go-api`.
2. Run `source .env && kamal deploy`.

## Verification
- Confirm that the Kamal deployment process completes without errors and the Go API is accessible on the VPS.
