# Skill: Deploy Rails to VPS

## Goal
Deploy the Ruby on Rails application located in the `rails8` directory to a Hetzner VPS using Kamal.

## Instructions
1. Change directory to `rails8`.
2. Run `source .env && kamal deploy`.

## Verification
- Confirm that the Kamal deployment process completes without errors and the application is accessible on the VPS.
